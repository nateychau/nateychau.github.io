Key

UCSD - La Jolla, CA | April 2019 - June 2019

In collaboration with: Jonathan Fong, Joseph Wang

Key is a full-stack web application that was built for the purpose of connecting freelance piano instructors with students. The project was mostly a learning experience, and our main goal with it was to integrate Google Maps' API to enable setting and searching by location. I'll be honest - its not our prettiest work, but you can check out the app here, and find its code on Github.

Front-end: HTML, Javascript/jQuery, CSS, Bootstrap

Back-end: Google Firebase, Google Maps API
